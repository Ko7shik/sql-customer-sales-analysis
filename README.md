# SQL Customer Sales Analysis

This project analyzes customer purchasing behavior and revenue performance using SQL JOINs across multiple tables.

## Dataset
- customers.csv
- products.csv
- orders.csv

## Key Business Questions
- Who are the highest revenue-generating customers?
- Which regions contribute the most revenue?
- Which products drive maximum revenue?

## Skills Demonstrated
- SQL JOINs (multi-table analysis)
- Revenue and customer analytics
- Business-driven SQL querying
- Analytical reasoning
